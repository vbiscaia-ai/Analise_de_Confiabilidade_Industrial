Projeto: Modelo de Confiabilidade Industrial Orientado à Decisão

Desenvolvi um modelo analítico de confiabilidade industrial a partir de 102 mil registros brutos de alarmes industriais.

O projeto foi estruturado em três camadas:

• ETL em Python para limpeza e padronização
• Modelagem dimensional (Star Schema) em MySQL
• Camada analítica GOLD com indicadores consolidados

Foram construídos indicadores robustos de:

MTTR (modelo conservador e amplo)

MTBF

Disponibilidade operacional

Frequência de falhas

Além do cálculo das métricas, o foco foi avaliar qualidade dos dados e estruturar um modelo confiável para tomada de decisão.

O projeto evoluiu de métricas técnicas para uma análise estratégica orientada à priorização de ativos, permitindo:

Identificação de máquinas críticas

Estruturação de ranking de risco operacional

Apoio à decisão de manutenção preventiva

Avaliação de impacto operacional por grupo técnico

Resultado: transformação de dados de alarmes em direcionamento estratégico para redução de indisponibilidade.
